Flow-Fund (originally Tracking Toll) is a state-of-the-art budget tracker designed to feel like a high-end intelligence dashboard. Featuring a dual-theme system that transitions seamlessly between a deep **"Neon Noir"** dark mode and a **"Premium Glassmorphism"** light mode, it completely revolutionizes the way you interact with your financial data.

## ✨ Core Features

- 🌓 **Adaptive Theming Engine**: Switch instantly between the deep "Neon Noir" cyber-aesthetic and a stunning, translucent glassmorphism Light Theme.
- 📈 **Interactive Expenditure Trajectory**: Beautiful area charts powered by Recharts that project your daily burn rate and forecast end-of-month spending.
- 🧮 **Categorical Intelligence Nodes**: Manage and visualize your budget allocations in real-time across various sectors (Food, Software, Transport, etc.).
- ⏱ **Timeline & Milestones**: Keep track of not just your expenses, but important financial milestones, notes, and reminders in a unified timeline.
- 💾 **Local Intelligence Persistence**: All financial data is securely processed and saved instantly in your browser's local storage.
- ⚡ **Lightning Fast UI**: Built on Vite + React + TypeScript, ensuring instant loading and flawless 60 FPS transitions.

## 🚀 Getting Started

Ensure you have [Node.js](https://nodejs.org/) installed on your machine.

### 1. Clone the Repository

```bash
git clone https://github.com/pixl6/Flow-Fund.git
cd Flow-Fund
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Boot the Intelligence Dashboard

```bash
npm run dev
```

The application will launch on `http://localhost:5173/`. 

## 🛠️ Tech Stack

- **Framework**: React 18
- **Language**: TypeScript
- **Styling**: Tailwind CSS (with highly customized CSS variable overrides)
- **Data Visualization**: Recharts
- **Icons**: Lucide React / Material Symbols
- **Build Tool**: Vite

## 🎨 Design Philosophy

Unlike standard spreadsheet-like budgeting apps, Flow-Fund was built to make managing money feel powerful. The dashboard borrows aesthetics from command centers, using glowing typography, translucent glass panels, and interactive grid lines to create an immersive user experience.
